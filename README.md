# Doctor Booking Application (MERN)

A full-stack doctor appointment booking system built using the **MERN stack**.  
Patients book appointments. Doctors manage schedules. Admins control the platform.

Simple idea. Real-world structure.

---

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Context API / Redux (optional)
- Tailwind CSS / CSS / MUI

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt

---

## Features

### Authentication
- User registration & login
- JWT-based authentication
- Role-based authorization (Patient / Doctor / Admin)

### Patient
- View doctors
- Book appointments
- View appointment history
- Cancel appointments

### Doctor
- Apply for doctor account
- Manage availability
- View appointments
- Update appointment status

### Admin
- Approve or reject doctors
- Manage users and doctors
- View all appointments
